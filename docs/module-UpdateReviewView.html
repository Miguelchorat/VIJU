Module: UpdateReviewView

Componente para editar una reseña de videojuego.

Data

Name Type Default value Description
title string - El título de la reseña.
message string - El cuerpo de la reseña.
score string - La puntuación asignada al videojuego.
videogame string - El videojuego de la reseña.
errorTitle string - Mensaje de error para el título de la reseña.
errorMessage string - Mensaje de error para el cuerpo de la reseña.
errorScore string - Mensaje de error para la puntuación de la reseña.
errorVideogame string - Mensaje de error para el videojuego de la reseña.
focusTitle boolean - Indica si el campo de título tiene el foco.
focusMessage boolean - Indica si el campo de cuerpo tiene el foco.
focusScore boolean - Indica si el campo de puntuación tiene el foco.
focusVideogame boolean - Indica si el campo de videojuego tiene el foco.
games Array - La lista de videojuegos disponibles.
API_GAMES string - La URL de la API para obtener la lista de videojuegos.
API_REVIEW string - La URL de la API para obtener la reseña actual.
API_UPDATE_REVIEW string - La URL de la API para actualizar la reseña.
scoreValidation RegExp - Expresión regular para validar la puntuación de la reseña.

Events

Name Payload Type Description
callAPI Llama a la API del servidor para encontrar la reseña del videojuego que incluyan la busqueda para modificar
callApiGames Pide al servidor que le devuelva todos los videojuegos por su nombre
updateReview Modifica una review con los datos proporcionados y lo envia al servidor
checkFields Valida los campos del formulario y actualiza la reseña si son correctos
