Module: Login

Componente que representa el formulario de inicio de sesión de un usuario.

Props

Name Type Default value Required? Description
menu Boolean - Yes Indica si el menú de la aplicación está activo o no.
account Boolean - Yes Indica si la cuenta del usuario está activa o no.
session Boolean - Yes Indica si la sesión del usuario está activa o no.
userId Number - Yes ID del usuario que está iniciando sesión.

Data

Name Type Default value Description
email String - Correo electrónico del usuario para iniciar sesión.
password String - Contraseña del usuario para iniciar sesión.
errorEmail String - Mensaje de error para el correo electrónico.
errorPassword String - Mensaje de error para la contraseña.
errorSession String - Mensaje de error para la sesión del usuario.
visibility Boolean - Indica si la contraseña es visible o no.
focusEmail Boolean - Indica si el campo de correo electrónico está enfocado o no.
focusPassword Boolean - Indica si el campo de contraseña está enfocado o no.
API_LOGIN String - URL de la API para iniciar sesión.
emailValidation RegExp - Expresión regular para validar correos electrónicos.
passwordValidation RegExp - Expresión regular para validar contraseñas.

Events

Name Payload Type Description
changeSession function Evento que se emite cuando la sesión del usuario cambia.
changeUserId function Evento que se emite cuando el ID del usuario cambia.
checkUser function Evento que se emite para verificar al usuario.
listenAccount function Evento que se emite para escuchar la cuenta del usuario.
listenMenu function Evento que se emite para escuchar el menú de la aplicación.
