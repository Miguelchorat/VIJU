Module: Signin

Componente para el registro de usuarios

Props

Name Type Default value Required? Description
menu boolean - Yes Propiedad que se utiliza para ocultar el menú de navegación

Data

Name Type Default value Description
email String - Correo electrónico del usuario
password String - Contraseña del usuario
repeatPassword String - Contraseña del usuario que se debe repetir para validarla
username String - Nombre de usuario del usuario
date String - Fecha de nacimiento del usuario
errorEmail String - Mensaje de error en caso de que el correo electrónico no sea válido
errorPassword String - Mensaje de error en caso de que la contraseña no sea válida
errorRepeatPassword String - Mensaje de error en caso de que las contraseñas no coincidan
errorUsername String - Mensaje de error en caso de que el nombre de usuario no sea válido
errorDate String - Mensaje de error en caso de que la fecha no sea válida
remember Boolean - Indicador de si el usuario desea que se recuerde su sesión
focusUsername Boolean - Indicador de si el input de nombre de usuario tiene el foco
focusEmail Boolean - Indicador de si el input de correo electrónico tiene el foco
focusPassword Boolean - Indicador de si el input de contraseña tiene el foco
focusRepeatPassword Boolean - Indicador de si el input de repetir contraseña tiene el foco
focusDate Boolean - Indicador de si el input de fecha de nacimiento tiene el foco
visibility Boolean - Indicador de si se está mostrando la contraseña
visibilityRepeat Boolean - Indicador de si se está mostrando la contraseña de repetición
API_REGISTER String - URL del endpoint para registrar usuarios
emailValidation RegExp - Expresión regular para validar el correo electrónico
passwordValidation RegExp - Expresión regular para validar la contraseña
usernameValidation RegExp - Expresión regular para validar el nombre de usuario
MIN_AGE_LIMIT Number - Edad mínima permitida para el registro
MAX_AGE_LIMIT Number - Edad máxima permitida para el registro

Events

Name Payload Type Description
listenRemember Cambia el valor de remember
listenVisibility Cambia el valor de visibility
listenVisibilityRepeat Cambia el valor de visibilityRepeat
async registerAccount - Registra un nuevo usuario en el servidor
checkFields Valida los campos del formulario y registra el usuario si son correctos
