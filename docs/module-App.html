Module: App

Componente para padre donde se mostrará los diferentes componentes de la aplicación

Data

Name Type Default value Description
search string - Valor del campo de búsqueda
account boolean - Mostrará o no si el usuario está logeado
menu number - Contendrá que menu se desplegará
loading number - Mostrará una pantalla de carga o no de ser necesario
user string - Contiene el nombre de usuario que inicio sesión
userId number - Contiene el id del usuario que inicio sesión
session number - Comprobará si existe una sesión en las cookies
API_CHECK string - La URL de la API para obtener si el servidor esta encendido.
API_LOGIN string - La URL de la API para obtener la comprobación de si el usuario existe.
API_CLOSE string - La URL de la API para cerrar la sesión del usuario
API_USER string - La URL de la API para recoger el nombre de usuario

Events

Name Payload Type Description
listenInput void Cambia el estado de de la variable search por el valor que le llegue
checkSession void Cambia el estado de session por el nuevo valor
checkUserId void Cambia el userId en caso de estar logeado
listenMenu void Cambia el estado de menu por el nuevo valor
listenAccount void Cambia el estado de account a su estado opuesto
callAPI Llama a la API del servidor para comprobar el estado del servidor
closeSession Llama a la API del servidor para cerrar la sesión del usuario
checkSession Llama a la API del servidor para comprobar el estado de la sesión del usuario
checkUser Llama a la API del servidor para comprobar el nombre de usuario del usuario que tiene sesión iniciada
