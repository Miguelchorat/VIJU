Componente para actualizar la información del usuario

Name Type Default value Required? Description

Name Type Default value Description
password string - Contraseña del usuario
repeatPassword string - Repetición de la contraseña del usuario
date string - Fecha de nacimiento del usuario
errorPassword boolean - Indicador de error en el campo de contraseña
errorRepeatPassword boolean - Indicador de error en el campo de repetición de contraseña
errorDate boolean - Indicador de error en el campo de fecha de nacimiento
focusPassword boolean - Indicador de enfoque en el campo de contraseña
focusRepeatPassword boolean - Indicador de enfoque en el campo de repetición de contraseña
focusDate boolean - Indicador de enfoque en el campo de fecha de nacimiento
visibility boolean - Indicador de visibilidad de la contraseña
visibilityRepeat boolean - Indicador de visibilidad de la repetición de contraseña
API_UPDATE_USER string - URL de la API para actualizar la información del usuario
emailValidation RegExp - Expresión regular para validar el formato del correo electrónico
passwordValidation RegExp - Expresión regular para validar el formato de la contraseña
usernameValidation RegExp - Expresión regular para validar el formato del nombre de usuario
MIN_AGE_LIMIT Date - Fecha mínima permitida para el campo de fecha de nacimiento (13 años atrás)
MAX_AGE_LIMIT Date - Fecha máxima permitida para el campo de fecha de nacimiento (120 años atrás)
