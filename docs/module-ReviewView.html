Module: ReviewView

Componente para mostrar una reseña individual.

Data

Name Type Default value Description
result Object - Los datos de la reseña obtenidos de la API.
submenu Boolean - Indica si el submenú está abierto o cerrado.
showSubmenu Boolean - Indica si el submenú debe mostrarse o no.
pathUpdate String - La ruta para actualizar la reseña.
API_REVIEW String - La URL para obtener los datos de la reseña.
API_DELETE String - La URL para eliminar la reseña.

Events

Name Payload Type Description
callAPI Llama a la API del servidor para encontrar la reseña del videojuego que incluyan la busqueda
listenSubmenu void Cambia el estado de submenu a su estado opuesto
checkUser void Busca la existencia del usuario que concuerde con la id de usuario en el servidor
checkPath void Comprueba el path en el que esta el usuario para enviarlo a un path diferente
deleteReview void Manda a eliminar la review del usuario por el id
