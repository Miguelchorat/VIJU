Module: CreateReviewView

Componente para crear una reseña.

Props

Name Type Default value Required? Description
title string - Yes Título de la reseña.
message string - Yes Mensaje de la reseña.
score string - Yes Puntuación de la reseña.
videogame string - Yes Nombre del videojuego relacionado con la reseña.
games Array.<Game> - Yes Lista de juegos para seleccionar.
API_GAMES string - Yes URL de la API para obtener la lista de juegos.
API_CREATE_REVIEW string - Yes URL de la API para crear una reseña.

Data

Name Type Default value Description
errorTitle string - Mensaje de error para el título de la reseña.
errorMessage string - Mensaje de error para el cuerpo de la reseña.
errorScore string - Mensaje de error para la puntuación de la reseña.
errorVideogame string - Mensaje de error para el videojuego de la reseña.
focusTitle boolean - Indica si el input del título tiene el foco.
focusMessage boolean - Indica si el input del cuerpo de la reseña tiene el foco.
focusScore boolean - Indica si el input de la puntuación de la reseña tiene el foco.
focusVideogame boolean - Indica si el input del videojuego tiene el foco.
scoreValidation RegExp - Expresión regular para validar la puntuación.

Events

Name Payload Type Description
callApiGames Pide al servidor que le devuelva todos los videojuegos por su nombre
createReview Crea una review con los datos proporcionados y lo envia al servidor
checkFields Valida los campos del formulario y registra la reseña si son correctos
